js设置方法如下
示例:<SCRIPT LANGUAGE="JavaScript" src="js.php?fid=76&type=lastview&icon=1&target=1"></SCRIPT>
请把以上代码按你的需要修改后,即可插入网页适当的地方调取显示数据
其中:
fid=76 你可以改成你要显示的栏目FID,如果想同时显示多个栏目的话,请用,号隔开,比如fid=1,2,33,44
type=lastview 你可以改成type=hot或type=com或type=new 其中lastview代表码最近浏览,hot代表最热,com代表推荐,new代码最新
icon=1 你可改成icon=0 其中1代表使用小图标,0代表不使用标题图标
target=1 你可改成target=0 其中1代表新开窗口,0代表不必新开窗口
注意:如果给其他模块或系统调用的话.js.php前面必须加上真实路径.比如http://or58.com/js.php
